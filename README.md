# Tarkov Stream Producer

![Coverage](https://img.shields.io/badge/Coverage-%25-red)

A Rust application for tracking Escape from Tarkov statistics and providing a REST API for stream overlays and automation.

## Features
- **Session Tracking:** Monitor performance across different stream sessions.
- **Raid Analytics:** Track state transitions, kills, and survival rates.
- **REST API:** Axum-based API for manual or automated control.
- **TDD Driven:** High test coverage and robust error handling.

## Development
- **Language:** Rust
- **Database:** SQLite (via sqlx)
- **API Framework:** Axum
- **Coverage:** Tarpaulin
